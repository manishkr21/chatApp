for backend : 
  open terminal in node server

```
  1. npm init
  2. npm i socket.io
  3. npm install -g nodemon # or using yarn: yarn global add nodemon
  4. Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
  5. nodemon .\index.js
  6. app started
```
for frontend:
 at index.html 
```
python3 -m http.server 5500
```
